Connection
==========

> `connection`객체는 기기의 무선 전화나 와이파이 연결 정보의 접근을 허가한다.

이 객체는 navigator.network 인터페이스에 접근된다.

Properties
----------

- connection.type

Constants
---------

- Connection.UNKNOWN
- Connection.ETHERNET
- Connection.WIFI
- Connection.CELL_2G
- Connection.CELL_3G
- Connection.CELL_4G
- Connection.NONE
